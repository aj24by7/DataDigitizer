DATA DIGITIZER 2.11 WINDOWS 11 INSTALL MANUAL

This package is for Windows 11.

You do not need Python.
You do not need VS Code.
You do not need to install Tesseract.
You do not need to use GitHub Desktop.

The easiest installer is:

DataDigitizer-2.11-Windows11-Installer.exe

It opens a small installer window with a progress bar. It installs:

- Digitizer
- AccuracyTester
- DataDigitizer CLI


OPTION A - EASIEST INSTALL

1. Download DataDigitizer-2.11-Windows11-Installer.exe from the GitHub release page.

2. Open your Downloads folder.

3. Double-click DataDigitizer-2.11-Windows11-Installer.exe.

4. If Windows shows "Windows protected your PC", click More info.

5. Click Run anyway.

6. The Data Digitizer installer window opens.

7. Click Install.

8. Wait for the progress bar to finish.

9. Click Close.

10. Look on your Desktop.

You should now see:

- Digitizer
- AccuracyTester
- DataDigitizer CLI

Double-click Digitizer to open the normal app.


OPTION B - ZIP DOWNLOAD

1. Download DataDigitizer-2.11-Windows11.zip from the GitHub release page.

2. Right-click the ZIP file.

3. Click Extract All.

4. Click Extract.

5. Open the extracted folder.

6. Double-click DataDigitizer-2.11-Windows11-Installer.exe.

7. Click Install.

If you accidentally double-click Install-Windows11.bat, it will open the same installer window when the installer exe is present.


OPTION C - NO INSTALLER, JUST RUN THE APP

1. Download DataDigitizer-2.11-Windows11.zip.

2. Right-click the ZIP file.

3. Click Extract All.

4. Open the extracted folder.

5. Double-click digitizer.exe.

That opens the normal Digitizer app.


HOW TO PIN TO TASKBAR

1. Open Digitizer.

2. Look at the Windows taskbar at the bottom of the screen.

3. Right-click the Digitizer icon.

4. Click Pin to taskbar.

Now Digitizer stays on the taskbar like Chrome or VS Code.


IF WINDOWS BLOCKS THE APP

Windows may show a message because this is not Microsoft Store software.

If you see "Windows protected your PC":

1. Click More info.

2. Click Run anyway.

If the file will not open after download:

1. Right-click the exe or zip file.

2. Click Properties.

3. Check Unblock if you see it.

4. Click Apply.

5. Try again.


WHAT EACH FILE DOES

DataDigitizer-2.11-Windows11-Installer.exe
Recommended installer. It opens a small installer window with a progress bar.

digitizer.exe
Normal app. Double-click this for the graph digitizer.

accuracytester.exe
Separate accuracy testing app.

DataDigitizer-2.11.exe
Command-line version for advanced users.

Install-Windows11.bat
Fallback launcher. Normal users should use DataDigitizer-2.11-Windows11-Installer.exe.


CLI EXAMPLE

Open PowerShell in the extracted folder and run:

.\DataDigitizer-2.11.exe 'digitizer_cli(pic_dir="C:\Users\User\Pictures\Screenshots\Example 2.png", output_dir="C:\Users\User\Downloads\testcli")'

You can copy the image path from File Explorer:

1. Right-click the image.
2. Click Copy as path.
3. Paste it after pic_dir=.

The CLI prints something like:

Pipeline done. Output data to C:\...\plot_digitized_points.csv. Points: 201.
Overlapping plot: C:\...\plot_digitized_overlay.png
