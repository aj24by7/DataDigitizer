DATA DIGITIZER 2.11 WINDOWS 11 INSTALL MANUAL

This package is for Windows 11.

You do not need Python.
You do not need VS Code.
You do not need to install Tesseract.
You do not need to use GitHub Desktop.

The apps are self-contained exe files.

IMPORTANT DOWNLOAD NOTE

The easiest and safest download is the release ZIP:

DataDigitizer-2.11-Windows11.zip

Do not use the green Code button unless you know what source code is.

If you accidentally downloaded a folder named DataDigitizer-main, the installer can now fix that by downloading the missing exe files from the official GitHub release. The computer must be connected to the internet.


OPTION A - EASIEST INSTALL

1. Download DataDigitizer-2.11-Windows11.zip from the GitHub release page.

2. Open your Downloads folder.

3. Right-click DataDigitizer-2.11-Windows11.zip.

4. Click Extract All.

5. Click Extract.

6. Open the extracted folder.

7. Double-click Install-Windows11.bat.

8. If Windows asks for permission, click Run.

9. Wait for the installer to say Data Digitizer 2.11 installed successfully.

If it says it is downloading digitizer.exe, accuracytester.exe, or DataDigitizer-2.11.exe, let it finish. That means you opened the installer from the source-code folder, and it is automatically fixing the missing files.

10. Close the installer window.

11. Look on your Desktop.

You should now see:

- Digitizer
- AccuracyTester
- DataDigitizer CLI

Double-click Digitizer to open the normal app.


OPTION B - NO INSTALLER, JUST RUN THE EXE

1. Download DataDigitizer-2.11-Windows11.zip.

2. Right-click the zip file.

3. Click Extract All.

4. Open the extracted folder.

5. Double-click digitizer.exe.

That opens the normal Digitizer app.

You can also double-click:

- accuracytester.exe
- DataDigitizer-2.11.exe


HOW TO PIN TO TASKBAR

1. Open Digitizer.

2. Look at the Windows taskbar at the bottom of the screen.

3. Right-click the Digitizer icon.

4. Click Pin to taskbar.

Now Digitizer stays on the taskbar like Chrome or VS Code.


IF WINDOWS BLOCKS THE APP

Windows may show a message because this is not Microsoft Store software.

If you see "Windows protected your PC":

1. Click More info.

2. Click Run anyway.

If the file will not open after download:

1. Right-click the zip file or exe file.

2. Click Properties.

3. Check Unblock if you see it.

4. Click Apply.

5. Try again.


WHAT EACH FILE DOES

digitizer.exe
Normal app. Double-click this for the graph digitizer.

accuracytester.exe
Separate accuracy testing app.

DataDigitizer-2.11.exe
Command-line version for advanced users.

Install-Windows11.bat
Creates Desktop and Start Menu shortcuts.


CLI EXAMPLE

Open PowerShell in the extracted folder and run:

.\DataDigitizer-2.11.exe "digitizer_cli(pic_dir='C:/path/to/plot.png', output_dir='')"

The CLI prints something like:

Pipeline done. Output data to C:\...\plot_digitized_points.csv. Points: 201.
Overlapping plot: C:\...\plot_digitized_overlay.png
